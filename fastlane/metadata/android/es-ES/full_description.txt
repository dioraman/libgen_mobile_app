Este es un cliente móvil no oficial para <a href="http://gen.lib.rus.ec">Library Genesis</a>.

El proyecto es completamente independiente y Open Source. Su desarrollador
no mantiene ningún vínculo con Library Genesis.

Toda colaboración es bienvenida.
Revisa el <a href="https://github.com/manuelvargastapia/libgen_mobile_app">repo</a>.

Funcionalidades:

* Buscar libros por título, autor u otros campos
* Ver información detallada del libro (páginas, tamaño, etc)
* Descargar libro (directo al directorio de Descargas)
* Escoger entre tema Claro y Oscuro
* Idiomas: inglés, español, polaco, portugués, alemán, japonés, francés y chino
* La app provee acceso a las secciones Sci-Tech y Fiction

El único permiso requerido es <b>Almacenamiento</b>. Se usa sólo para depositar los archivos descargados en el directorio de Descargas.
Hecho con <a href="https://flutter.dev">Flutter</a>.
